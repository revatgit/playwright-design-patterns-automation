# Playwright Automation Framework (Design Patterns)

This project demonstrates a Playwright automation framework built using TypeScript and modern testing best practices.

## 🚀 Features

- Page Object Model (POM)
- Custom Fixtures for reusable setup
- TypeScript for type safety
- Modular test structure
- Scalable automation design

## 🧰 Tech Stack

- Playwright
- TypeScript
- Node.js

## 📂 Project Structure
tests/ → Test cases
pages/ → Page Object Models
fixtures/ → Custom fixtures
utils/ → Test data and helpers
